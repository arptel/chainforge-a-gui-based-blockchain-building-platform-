# Certificate Verification Demo - ChainForge

This project is a 3-tier demonstration of the ChainForge platform's capabilities using Python Smart Contracts.

## Directory Structure
- `/chainforge_node` - The patched, running blockchain network generated by ChainForge.
- `/backend` - A FastAPI middleware server handling JWT Auth and wrapping the blockchain REST calls.
- `/frontend` - A stunning React + Vite + Tailwind dashboard built for the demonstration.

## How to Run the Demo (Easy Way)
You can launch the entire ecosystem (all 3 services) simultaneously by double clicking or running the included PowerShell script:
```powershell
.\run_demo.ps1
```

## How to Run the Demo (Manual Way)

To run the entire demo system manually, you need to start 3 separate services. Open 3 different terminal windows:

### Terminal 1: Start the ChainForge Node
```bash
cd certificate_verification_system/chainforge_node
pip install -r requirements.txt
python main.py --api-port 8080 --port 5000
```
*(Wait 5-10 seconds for the genesis block to mine and the API server to start)*

### Terminal 2: Start the FastAPI Backend Middleware
```bash
cd certificate_verification_system/backend
pip install fastapi uvicorn pydantic pyjwt httpx
uvicorn main:app --reload --port 8001
```

### Terminal 3: Start the React Frontend UI
```bash
cd certificate_verification_system/frontend
npm install
npm run dev
```

## Demo Script / Flow
1. Open the frontend web app (usually `http://localhost:5173`).
2. Navigate to the **College Portal**.
3. Login using the default mock credentials:
   - Username: `college_a`
   - Password: `password123`
4. Use the dashboard to issue a new certificate for a student (e.g. "John Doe", "B.S. Computer Science").
5. Note the generated **Certificate ID** provided in the success message.
6. Open an incognito window or log out, and go to the **Company Verification Gateway** (Verify tab).
7. Paste the Certificate ID to verify that the blockchain validates the credential securely!
8. Share the unique student link so the 'student' can view their premium diploma visually.
