Flow Diagrams
Code Generation
When the user clicks Download, the builder reads the project config, walks the template directory, filters modules by config, transpiles Solidity contracts via LLM, and produces a self-contained ZIP.
Node Architecture
When you run python main.py, the Dependency Injector reads genesis.json and wires together the correct consensus, sync, VM, and network modules.
Transaction Lifecycle
A transaction passes through 7 stages before becoming permanent on-chain data. This applies to ALL transaction types: transfers, contract calls, and certificate operations.
PBFT Consensus
Practical Byzantine Fault Tolerance requires 3 communication rounds to finalize a block. Tolerates up to f = floor of (n-1)/3 Byzantine faults. Quorum = 2f + 1.
